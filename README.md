# Lava Rise

Two-player stick-fighter survival game for iPad. The lava rises. Climb or die. Knock your opponent in first.

**Players:** Ford (blue, left controls) vs Finn (red, right controls)  
**Platform:** Single iPad, landscape, touch only  
**Deployed:** https://theroller.github.io/lava-rise

## How to Play

- Lava rises from the bottom — keep climbing
- Jump to higher platforms to survive
- Attack to knock your opponent toward the lava
- Last one alive wins

## Controls

| | Ford (left) | Finn (right) |
|---|---|---|
| Move | ← → | ← → |
| Jump | ↑ | ↑ |
| Attack | ⚔ | ⚔ |

Axe weapon can break cracked (brown) platforms.

## Running Locally

Open `index.html` in a browser. Keyboard fallback: Ford = WASD + F, Finn = Arrow keys + Enter.
